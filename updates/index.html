How updates work

Every installed copy of AlphaTill checks this page on launch and from Admin → Settings → Software updates → Check for updates. When a newer version is found, the app downloads and installs it for you — no reinstall, no lost data.

Tip: You only need to download manually for a brand-new machine. Existing tills update themselves from here.
